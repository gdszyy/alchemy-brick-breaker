# 炼金打砖块 (Alchemy Brick Breaker)

一款融合**弹珠台**与**打砖块**玩法的创意游戏，玩家通过选择弹珠、收集材料、配置魔药来对抗无尽的敌人。

## 游戏概述

### 核心玩法循环

游戏采用"无尽街机"模式。玩家通过以下循环进行游戏，直到敌人突破防线：

1. **命运抉择** - 选择3颗弹珠
2. **研磨与提纯** - 通过弹珠台收集材料
3. **战斗与发射** - 射击消除敌人
4. **敌人推进** - 敌人下压并生成新敌人

## 游戏阶段

### 第一阶段：命运抉择 (Phase 0: Selection)

每一轮开始前，系统随机提供5颗弹珠，玩家需从中选择3颗进入研磨阶段。

**弹珠类型与概率：**

- ⚪ **纯净弹珠** (White, 60%) - 无特殊效果
- 🎨 **同化弹珠** (Colored, 25%) - 撞击普通钉子时，将其转化为对应属性
- 🔴 **爆破弹珠** (Red Stripe, 10%) - 生成的魔药子弹具有爆炸效果
- 🌈 **七彩棱镜** (Rainbow, 5%) - 撞击第一颗钉子后分裂为两颗

### 第二阶段：研磨与提纯 (Phase 1: Gathering)

通过类似"柏青哥/弹珠台"的玩法收集材料，构建魔药配方。

**属性钉子：**

- 🔵 **基底** (Base) - 增加基础伤害
- 🟢 **弹性** (Bounce) - 增加子弹反弹次数
- 🔴 **穿透** (Pierce) - 增加子弹穿透敌人次数
- 🟡 **散射** (Scatter) - 增加额外子弹数量
- 🟣 **增幅** (Damage) - 大幅增加伤害

**特殊槽：**

- 🌌 **回溯** (Recall) - 弹珠保留X轴位置，Y轴传送回顶部
- ⚡ **连射** (Multicast) - 最终子弹发射次数 +2
- ⑂ **分裂** (Split) - 弹珠分裂为两个
- 🟢 **变大** (Enlarge) - 弹珠体积增大，更容易碰撞钉子

### 第三阶段：战斗与发射 (Phase 3: Combat)

使用收集到的魔药配方，以"打砖块"的方式攻击敌人。

**射击规则：**

- 拖拽瞄准，松开射击
- 发射角度与水平线的夹角必须 > 20度
- 子弹存在时间限制（15秒）

**子弹特性：**

- 碰撞优先级：穿透 > 反弹
- 屏幕四周均为反弹墙
- 爆破弹珠生成的子弹击中敌人产生范围伤害

## 计分系统

- **基础分** - 击杀敌人获得等同于其初始血量的分数
- **连击倍率** - 初始倍率 x1.0，每击杀一个敌人 +0.2，每次新发射时重置

## 文件说明

- `index.html` - 游戏主程序文件

## 如何运行

1. 在浏览器中打开 `index.html` 文件
2. 按照游戏提示进行操作
3. 选择弹珠 → 收集材料 → 射击敌人 → 重复循环
